---

### 📌 详细说明 (Detailed Description)

**DevHelper** 是一款专为研发、测试与产品经理量身定制的“一站式智能开发者工具箱”。我们深度整合了 16 个日常高频使用的开发辅助工具，并首创了**内置本地离线 AI 助手**。不仅极大地减少了开发过程中的“工具切换”成本，更为您的敏感代码和数据提供了 100% 的隐私安全保障。

---

#### 🌟 核心特色 🌟

1. **🔒 100% 本地离线 AI (Chrome Built-in AI)**
   - 智能联动：内嵌于 JSON 格式化、JWT、编解码等多款工具中，提供一键结构解读、漏洞排查与优化建议。
   - 隐私无忧：基于本地离线大模型推理，无需联网，无需配置任何 API Key，企业级敏感代码/数据绝不外泄。

2. **📝 智能 Markdown 编辑器 (Markdown & Mermaid)**
   - 支持完整的 GitHub GFM 语法、代码高亮、表格与 Emoji。
   - 内置 **Mermaid.js** 支持，支持直接绘制流程图、时序图、甘特图等。
   - 具备 **双栏同步滚动**（智能检测活动源），导出时自动以首个 H1 标题作为文件名，提供免工具栏的纯净排版打印预览。

3. **📊 极速 JSON 工具链**
   - **JSON 格式化**：支持完美美化、极致压缩、强校验及双向转义。
   - **JSON 对比 (Diff)**：支持两段 JSON 文本的行级与字符级差异可视化对比，由 AI 精确总结修改逻辑。

4. **⚡ 16 个高频工具全集成**
   - **编解码**：URL、Base64、Unicode、Hex 一键互转，AI 自动检测编码格式。
   - **JWT 解析**：轻量化拆分 Token，检测有效期及签名完整度，内置 AI 安全评估。
   - **研发测试**：API 调试器（支持自定义 Header/请求体）、时间戳转换、Cron 表达式生成与 Regex 正则测试。
   - **快捷生成**：批量 UUID (v4)/雪花 ID/NanoID 生成器、高强度密码生成器、二维码生成与解析、图片 Base64 转换。

5. **🎨 现代极简美学 UI**
   - 采用精致的暗色毛玻璃与渐变设计，支持一键切换深色/浅色主题。
   - 极致的响应式排版与微动画交互，带来殿堂级的视觉体验。
