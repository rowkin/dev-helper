# DevHelper 更新日志

## v1.3.0 (2026-06-29)

### 🚀 新功能
*   **颜色转换工具深度扩展**：
    *   **屏幕级吸色 (EyeDropper)**：正式支持 Chrome 原生 `EyeDropper` API。点击“滴管拾色”后可触发高精度放大镜，支持吸取当前网页、其他浏览器标签页、甚至浏览器窗口外的桌面任意像素颜色。

### ⚙️ 优化与修复
*   **AI 表格流式渲染增强**：
    *   重构 `renderInlineMarkdown` 函数，增加占位符提取和流式表格逐行扫描识别机制，完美解决 AI 面板生成 Markdown 表格无法渲染、换行导致布局塌陷的问题。
*   **Gemini Nano 报错修复**：
    *   针对 LanguageModel 实例创建时 expectedOutputLanguages 对 'zh' 的类型抛错问题进行修复，对齐 Chrome 最新标准，消除控制台警告。
*   **版本同步**：
    *   全局同步更新 `manifest.json`、`package.json`、Popup 及全景控制台的版本信息为 `v1.3.0`。

## v1.2.0 (2026-06-29)

### 🚀 新功能
*   **新增「页面性能检测」工具 (⌥5)**：
    *   **核心指标监控**：通过 `PerformanceObserver` 及 `performance.timing` 接口，深度监控目标页面的首次绘制 (FP)、首次内容绘制 (FCP)、DOMContentLoaded (DOM就绪) 及 Load Time (完全加载) 指标。
    *   **网络时序瀑布图**：使用高对比度的 HSL 渐变色瀑布时序条，精准还原重定向、DNS查询、TCP握手、SSL握手、TTFB (首字节响应)、内容下载等各个阶段的相对时序耗时。
    *   **智能性能体检评分**：基于 Web Vitals 规范阈值，以精美的环状进度图为核心指标实时评分。
    *   **定制化调优方案**：内置算法针对 DNS 延迟、握手时间、服务器相应 (TTFB)、DOM 阻塞及内存泄漏提供定制化调优方案清单。
    *   **AI 智能专家诊断**：无缝对接 DevHelperAI 助手。在性能页面中一键唤醒 AI，自动传输时序 JSON 数据包，由大模型为您深度剖析网页加载瓶颈，给出代码及服务器配置级别的具体优化方案。

### ⚙️ 优化与修复
*   **导航系统升级**：
    *   在 Popup 快捷菜单与全景控制面板 (`index.html`) 中正式注册该功能，并将内置工具总数更新至 17 个。
    *   优化 `popup.js`，用户在被测页面中打开扩展时，会自动记录并携带当前活动页的 Tab ID 参数，实现“一键快速分析”。
*   **工程规范对齐**：
    *   全局同步更新 `manifest.json`、`package.json`、Popup 版权栏及设置项中的版本号为 `v1.2.0`。
    *   所有新增页面及事件逻辑均符合 Chrome Extension MV3 及 CSP 安全规范，禁止使用任何内联脚本。
