# DevHelper 更新日志

## v1.5.0 (2026-07-16)

### 🚀 v1.5.0 新功能

* **万能计算器 (Universal Calculator) 隆重上线**：
  * **标准与科学计算器**：基于无 `eval` 的 Shunting-yard 算法与 RPN 栈求值引擎，完全符合 Chrome MV3 极其严格的 Content Security Policy (CSP) 安全规范；支持隐式乘法自动补全（如 `3(4+5)` 自动解析为 `3*(4+5)`）；提供三角函数（角度制）、对数（ln, log）以及平方根等科学运算。
  * **实时汇率换算**：接入免费开放的 Exchange Rates API。内置了完备的降级与离线备份方案——在网络异常或接口受限时，自动加载 `chrome.storage.local` 历史缓存汇率，若缓存亦不可用则降级为出厂硬编码备份汇率，保证转换服务的 100% 连续性；提供以 100 CNY 为基准的常见币种双向对比表格与双向金额折算。
  * **房贷月供推演**：支持“等额本息”与“等额本金”两种经典还款路径的推演；支持自定义还款期数；自动计算首月月供、每月递减、总还款利息及总还款额，并生成逐期本息递减、待还剩余款项的详细还款明细推演表。
  * **AI 财务与数学智能诊断**：与 AI 智能解读抽屉深度绑定。支持对输入的数学公式含义进行讲解，科普币种宏观经济与跨境消费贴士，并能基于房贷利息差额、利率杠杆为用户提供智能的个人财务建议和提前还贷策略分析。

### ⚙️ v1.5.0 优化与修复

* **AI 助手本地 Mock 仿真升级**：
  * 针对本地双击预览调试等缺失 `chrome` 原生 API 的脱机环境，重构了 `js/ai-service.js` 的 `fetch` 拦截 Mock 数据，增加了对“计算器/汇率/房贷”多面板上下文的感应拦截。现在在本地非插件状态预览时，AI 助手也能返回针对当前计算面板的高质量模拟诊断报告，消除了原有通用闲聊词的突兀感。
* **版本与工具全局升级**：
  * 升级 `manifest.json`、`package.json`、Popup 主页及关于面板的版本号至 `v1.5.0`，工具计数由 20 升级至 21，完成新一轮发布资产的封装。

## v1.4.1 (2026-07-08)

### 🚀 v1.4.1 新功能

* **图片放大查看器 (Lightbox)**：
  * **高拟真手势交互**：支持对放大后的图片进行鼠标拖拽平移及鼠标滚轮无限缩放，便于定位查看长截图中的微小细节。
  * **沉浸式控制面板**：基于毛玻璃特效 (Glassmorphism Modal)，在顶部悬浮微型控制栏，支持放大、缩小、1:1 复位、顺时针 90 度旋转、以及一键下载/保存大图文件。
* **常驻 AI 助手删除能力升级**：
  * **历史对话单项删除**：左侧历史对话列表增加了悬浮触发的删除按钮，删除当前对话后会自动向后加载或重新开启一个干净的会话。
  * **单个气泡精准删除**：每个聊天气泡右上角均新增了微型悬浮删除小按钮，点击会从本地存储中彻底剔除并更新整个聊天流的索引序列。

### ⚙️ v1.4.1 优化与修复

* **大资产缓存持久化优化 (解决截图刷新丢失)**：
  * 引入了基于 `mediaAssets` 的对话大图 Base64 字典机制。当生成二维码或长截图时，在本地自动转换为带时间戳后缀的唯一 ID 并存储在对应会话中。
  * 刷新页面加载历史时，大图可以完美持久化呈现，且在同一会话中多次截图/生成二维码之间独立渲染，互不覆盖。
* **工具集 Manifest 深度对齐**：
  * 重新整理并全量补充了 DevHelper 插件内实际上已完整搭载的全部 20 大内置开发者工具（如 API 接口测试、代码美化、正则测试等）。
  * 将 20 大工具完整注入 System Prompt 的静态上下文，并且同步更新了 Mock AI 对话大模型的功能列表展示与介绍逻辑。
* **版本全局对齐**：
  * 全局同步升级 `manifest.json`、`package.json`、Popup 页脚、设置面板及 site 落地页的版本号至 `v1.4.1`，工具总数同步更新至 `20`。

## v1.4.0 (2026-07-07)

### 🚀 v1.4.0 新功能

* **新增「IP 与网速检测」工具 (⌥6)**：
  * **IP 地址智能探测**：支持基于 WebRTC 的本地局域网 IP (Local IP) 智能穿透探测，以及通过安全公网 API 动态识别外网 IP (Public IP)、ISP 运营商及所属国家/省份/城市等地理位置。
  * **高精度网络基准跑分**：具备毫秒级 Ping（延迟）与 Jitter（抖动）测试，并通过 CDN 及安全上传端点动态计算网页实时下载速率 (Download Speed) 与上传速率 (Upload Speed)。
  * **AI 智能网络状态体检**：无缝对接 AI 诊断面板，自动解析网速及延迟数据，由大模型给出智能优化、DNS调整及网络异常诊断意见。
  * **网速历史数据持久化**：使用 `chrome.storage.local` 本地存储机制，自动保存最近 30 条网速检测历史记录，方便跟踪网络质量波动趋势。

### ⚙️ v1.4.0 优化与修复

* **截图工具功能升级**：
  * 重构并优化全页面（长图滚动）截图的滚动捕获算法，解决部分动态滚动条或粘性定位元素干扰导致的长图遗漏问题。
  * 升级 AI 截图分析框组件样式，使用优雅的高度动态计算与局部滚动机制，消除在左侧画布过高时右侧大量空白的问题；全面支持 AI 诊断分析结果以 Markdown 格式高保真渲染展示。
* **常驻 AI 助手交互打磨**：
  * 修复常驻 AI 聊天输入框由于内容过多引起视口溢出的问题，自适应折行增加高度。
  * 新增机器人回复时的流式 Loading 骨架屏态与流畅打字机动效，提升人机协作沉浸感。
* **版本与数据全局对齐**：
  * 全局同步升级 `manifest.json`、`package.json`、Popup 主页、设置项及 site 落地页的版本号至 `v1.4.0`，工具计数同步升级至 `19`。

## v1.2.0 (2026-06-29)

### 🚀 v1.2.0 新功能

* **新增「页面性能检测」工具 (⌥5)**：
  * **核心指标监控**：通过 `PerformanceObserver` 及 `performance.timing` 接口，深度监控目标页面的首次绘制 (FP)、首次内容绘制 (FCP)、DOMContentLoaded (DOM就绪) 及 Load Time (完全加载) 指标。
  * **网络时序瀑布图**：使用高对比度的 HSL 渐变色瀑布时序条，精准还原重定向、DNS查询、TCP握手、SSL握手、TTFB (首字节响应)、内容下载等各个阶段的相对时序耗时。
  * **智能性能体检评分**：基于 Web Vitals 规范阈值，以精美的环状进度图为核心指标实时评分。
  * **定制化调优方案**：内置算法针对 DNS 延迟、握手时间、服务器相应 (TTFB)、DOM 阻塞及内存泄漏提供定制化调优方案清单。
  * **AI 智能专家诊断**：无缝对接 DevHelperAI 助手。在性能页面中一键唤醒 AI，自动传输时序 JSON 数据包，由大模型为您深度剖析网页加载瓶颈，给出代码及服务器配置级别的具体优化方案。

### ⚙️ v1.2.0 优化与修复

* **导航系统升级**：
  * 在 Popup 快捷菜单与全景控制面板 (`index.html`) 中正式注册该功能，并将内置工具总数更新至 17 个。
  * 优化 `popup.js`，用户在被测页面中打开扩展时，会自动记录并携带当前活动页的 Tab ID 参数，实现“一键快速分析”。
* **工程规范对齐**：
  * 全局同步更新 `manifest.json`、`package.json`、Popup 版权栏及设置项中的版本号为 `v1.2.0`。
  * 所有新增页面及事件逻辑均符合 Chrome Extension MV3 及 CSP 安全规范，禁止使用任何内联脚本。
