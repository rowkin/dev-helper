# DevHelper 隐私权政策

七九在线科技（以下简称“我们”或“79team”）非常重视用户隐私，并致力于保护您的个人数据。本隐私权政策旨在向您说明 DevHelper 浏览器扩展程序（以下简称“本扩展”）如何收集、使用和保护您的数据，以确保本扩展符合 Chrome Web Store 开发者政策要求。

---

## 一、我们收集与处理的数据

根据本扩展的功能设计，我们严格遵循**最小化数据收集原则**，大多数功能完全在本地进行：

### 1. 本地存储数据 (chrome.storage.local)
- **工具配置与历史记录**：我们会将您的工具使用首选项（如主题样式、排版模式、双栏同步滚动设置等）以及您主动保存的历史记录存储在您的本地浏览器中。
- **本地存储目的**：仅用于提升您的日常使用体验，让您无需重复配置工具。

### 2. 剪贴板数据 (clipboardRead / clipboardWrite)
- **快捷复制与粘贴**：当您使用 JSON 压缩/美化、时间戳转换、Markdown 编辑器等工具的“复制结果”或“快速导入剪贴板”按钮时，本扩展会读取或写入您的系统剪贴板。
- **读取目的**：仅在您主动点击相关操作时触发，用于将需要处理的文本或代码快速填入当前工具的输入区。

### 3. 活动标签页与脚本执行 (activeTab / scripting / tabs)
- **开发者工具调试**：本扩展需要获取当前活动标签页的信息，以提供注入辅助开发脚本等特定的调试功能。
- **用途**：仅用于分析当前页面的开发状态并为您呈现相关信息。

### 4. 远程 API 请求 (host_permissions)
- **AI 助手对话**：当您使用内嵌的 AI 助手或一键解读功能时，本扩展会直接请求您自定义配置的 AI 服务提供商（如 OpenAI、Google Gemini、Anthropic 等）。
- **静态 CDN 依赖**：为渲染 Markdown 工具中的 Mermaid.js 图表，本扩展可能会通过公共 CDN 请求相关的静态 JavaScript/CSS 资源。

---

## 二、数据用途与处理规则

1. **完全本地运行**：本扩展不具备任何中央云端服务器。您在所有开发工具（包括 JSON 工具、Markdown 编辑器等）中输入的任何文本、代码、凭证，均直接在您的本地浏览器沙箱中进行解析和渲染，**绝不会上传到我们的任何服务器**。
2. **AI API 密钥安全**：您在设置中填写的 API Key 和代理地址完全保存在您的本地浏览器存储（`chrome.storage.local`）中。当发起 AI 对话时，本扩展会以直连方式向大模型接口发送请求，**不会中转、截获或保存您的 API 密钥与任何提示词内容**。
3. **无追踪与无广告**：本扩展不包含任何广告追踪模块，也不使用任何分析工具（如 Google Analytics 等）来监测您的个人使用习惯。

---

## 三、数据的共享与转让 (Data Sharing and Transfer)

我们郑重承诺：
1. **绝不向任何第三方出售、转让、共享或披露您的任何数据**。
2. **所有数据不会被用于除本扩展功能之外的任何商业盈利目的**（如广告推送、个性化画像建立等）。
3. 所有本地存储数据受浏览器沙箱安全框架保护，未获授权的第三方网站或程序无法读取。

---

## 四、第三方服务的隐私政策

当您使用 AI 功能时，传输的提示词和代码片段将受您配置的第三方大模型服务商（例如 OpenAI, Google 等）隐私权政策约束。我们强烈建议您在使用前了解其对应的隐私声明。

---
